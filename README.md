# Agent Based Model Flight Service Simulation


- Agent based model for a cabin service simulation using NetLogo.
- Web Version [here](https://carlostussi.github.io/agent_model_777/)
- Goal: Determine the ideal amount of crew to serve passengers given different passenger load and cabin crew distribution.


## Basic Instructions
1) Click "setup" to confirm the parameter selection for the simulation.
2) Click "go" to run the simulation.
- You can select the total number of cabin crew by adjusting the "total-crew" sliding bar.
- You can select the total number of pax by adjusting the "total-pax" sliding bar.
- You can determine the patience level by selecting an option from the drop down box "patience-level".


** OBS: This model is still under construction and improvements and updates are made frequently.
